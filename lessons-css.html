CSS Basics

CSS (Cascading Style Sheets) controls the presentation of HTML elements. It lets you set colors, spacing, layout, and more.

Selectors

Selectors choose which HTML elements to style.

/* element selector */

Box Model

Every element is a box: content, padding, border, margin.

.box {
  width: 200px;
  padding: 16px;
  border: 2px solid #2563eb;
  margin: 12px;
}

This affects the total space the element occupies.

Layout with Flexbox

Flexbox makes aligning items easy:

.container {
  display: flex;
  gap: 16px;
  justify-content: center;
  align-items: center;
}

Example structure:

<div class="container">
  <div class="box">Item 1</div>
  <div class="box">Item 2</div>
</div>
